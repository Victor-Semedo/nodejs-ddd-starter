# Starter API
This is a project template.

## Technology Stack

* Typescript
* Nodejs
* MongoDB
* Mongoose

## Architecture

* Onion Architecture
* Domain Driven Design

## Running

### Pre-Conditions

* Node installed (Preferably version 14.9.0)
* Mongodb installed

To install all the dependencies run:
> npm i

To run the application
> npm run start
